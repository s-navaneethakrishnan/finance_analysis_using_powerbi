# 📊 Finance Analysis Using Power BI

An interactive Power BI dashboard that provides comprehensive financial insights through data visualization, KPI tracking, and business intelligence reporting. This project demonstrates end-to-end data analysis skills, including data cleaning, modeling, DAX calculations, and dashboard design.

---

## 📌 Project Overview

This project focuses on analyzing financial data to help stakeholders monitor business performance and make data-driven decisions.

The dashboard provides:

- Revenue Analysis
- Profit Analysis
- Sales Performance
- Product Performance
- Regional Analysis
- Customer Insights
- KPI Monitoring
- Interactive Filtering

---

## 🎯 Project Objectives

- Analyze financial performance using Power BI.
- Track key business KPIs.
- Identify high-performing products and regions.
- Compare sales and profit trends.
- Create an interactive dashboard for decision-making.

---

## 🛠 Tools & Technologies

- **Power BI Desktop**
